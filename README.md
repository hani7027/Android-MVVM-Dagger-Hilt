# Android-MVVM-Dagger-Hilt

Android login application with form validation and Database storage made using MVVM and Data binding.

## Libraries used

- Android Databinding
- Google Material Design
- Kotlin coroutines
- ViewModel
- LiveData
- Dagger Hilt
- Room Database
